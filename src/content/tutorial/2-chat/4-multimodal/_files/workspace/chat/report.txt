ACME Corp — Q3 2024 Quarterly Report

EXECUTIVE SUMMARY
Revenue for Q3 2024 reached $48.2M, up 22% year-over-year. All three product
lines grew, with Cloud Services leading at +41% YoY. Operating margin improved
to 18.4% from 14.1% in Q3 2023.

PRODUCT LINE PERFORMANCE

Cloud Services
  Revenue:  $21.4M  (+41% YoY)
  Customers: 1,840  (+29% YoY)
  Key driver: Enterprise tier adoption up 67%; new data-residency features
  launched in August drove EMEA deals.

Developer Tools
  Revenue:  $16.8M  (+15% YoY)
  Customers: 12,400 (+8% YoY)
  Key driver: VS Code extension reached 2M installs; CLI v3.0 release in
  September expanded paid conversions.

Professional Services
  Revenue:  $10.0M  (+7% YoY)
  Projects:  94 completed
  Key driver: Demand for migration engagements from on-prem to Cloud Services
  tier; average project size up 18%.

OUTLOOK
Q4 2024 guidance: $51–53M revenue. Full-year 2024 guidance raised to
$183–185M (previously $178–182M). Cloud Services expected to exit 2024
at $90M ARR run rate.
